Our Mission

To help individuals track their fitness journey and achieve their health goals.

Key Features:

  • Track workouts with details like type, duration, and calories burned.
  • Quickly find workouts by type.
  • Sort workouts by date using bubble sort.
  • Manage workouts using an ArrayList and prioritize them using a priority queue.
